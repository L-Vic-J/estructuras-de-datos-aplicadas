## Estructura de Datos Aplicadas

Este repositorio contiene el desarrollo y análisis de distintos escenarios prácticos orientados al uso de estructuras de datos fundamentales, como listas enlazadas, pilas y colas, aplicadas a contextos reales dentro del ámbito educativo, tecnológico y comunitario.

El objetivo del proyecto es comprender, implementar y justificar el uso de cada estructura de datos según las necesidades de cada sistema, fortaleciendo los conceptos vistos en el curso de Estructuras de Datos.

## Objetivos del Proyecto

Aplicar estructuras de datos lineales en sistemas reales.

Analizar la elección adecuada de cada estructura según el problema.

Implementar recorridos, inserciones y eliminaciones de datos.

Desarrollar pensamiento lógico y estructurado en la resolución de problemas.

## Escenarios Desarrollados

1. Sistema de Gestión de Biblioteca

Se desarrolla un sistema para la administración de una biblioteca educativa, donde se aplican las siguientes estructuras:

Lista enlazada simple para el catálogo de libros, permitiendo recorrer los títulos disponibles de forma ordenada.

Lista doble circular para la gestión de usuarios (estudiantes y docentes), facilitando la navegación hacia adelante y hacia atrás.

Pila asociada a cada usuario para almacenar el historial de préstamos, priorizando el libro más reciente.

Cola para gestionar la lista de espera de libros no disponibles, atendiendo las solicitudes en orden de llegada.

2️. Plataforma de Aprendizaje Virtual

Sistema orientado a la gestión de cursos y actividades académicas dentro de una plataforma educativa:

Lista simple circular para la navegación continua entre cursos.

Lista enlazada simple para organizar los módulos de cada curso de forma secuencial.

Pila para el manejo de actividades pendientes de los estudiantes, destacando la más reciente.

Cola para gestionar solicitudes de soporte técnico, respetando el orden de ingreso.

3️. Sistema de Gestión de Proyectos Colaborativos

Aplicación enfocada en el trabajo en equipo y el control de proyectos:

Lista doble circular para administrar los integrantes de cada proyecto.

Cola para almacenar y asignar tareas según su orden de creación.

Pila para registrar el historial de modificaciones del proyecto, permitiendo consultar los cambios más recientes.

Lista enlazada simple para organizar los hitos o entregables del proyecto de manera lineal.

4️. Aplicación para la Gestión de Reciclaje Comunitario

Proyecto orientado al impacto social y ambiental:

Lista doble circular para el registro de participantes del programa de reciclaje.

Lista simple circular para modelar las rutas continuas de los camiones recolectores.

Cola para gestionar las solicitudes de recolección de materiales.

Pila para almacenar el historial de entregas realizadas por cada participante.

Lista enlazada simple para el catálogo de materiales reciclables disponibles en la aplicación.
